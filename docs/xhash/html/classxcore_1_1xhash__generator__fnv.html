fnv_32a_buf - perform a 32 bit Fowler/Noll/Vo FNV-1a hash on a buffer

input: buf - start of buffer to hash len - length of buffer in octets hval - previous hash value or 0 if first call

returns: 32 bit hash as a static hash type

NOTE: To use the recommended 32 bit FNV-1a hash, use FNV1_32A_INIT as the hval arg on the first call to either fnv_32a_buf() or fnv_32a_str().
